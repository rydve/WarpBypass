**WarpBypass** — это инструмент для автоматизации сетевых соединений и оптимизации маршрутизации трафика. Утилита обеспечивает стабильный и отказоустойчивый доступ к глобальным ресурсам через протокол Cloudflare WARP, применяя методы глубокой инспекции и модификации пакетов для предотвращения разрывов соединения.

Разработано для обеспечения непрерывной сессии в рабочих средах, мессенджерах (вкл. Discord) и сервисах, критичных к потере сетевых пакетов.

## 🚀 Основные возможности

- **Автоматизированное развертывание:** Скрипт самостоятельно выполняет аудит системы, загрузку недостающих компонентов, настройку маршрутов и инициализацию служб.
- **Стабилизация соединения:** Интеграция продвинутых алгоритмов маскировки трафика на уровне сокетов для обхода ограничений магистральных провайдеров.
- **Скрытый режим работы (Headless):** Клиент Cloudflare WARP функционирует как фоновая системная служба, не требующая графического интерфейса и не отвлекающая пользователя.
- **Продвинутая конфигурация:**
    - **Автоматические пресеты:** Сохранение и мгновенный запуск предпочтительного профиля маршрутизации.
    - **Диагностика соединения (TCP Ping):** Встроенный модуль замера времени установки TCP-сессии до целевых узлов (отображает реальное время отклика сервиса, включая TCP Handshake).
    - **Автоматический откат изменений:** Механизм фонового мониторинга гарантирует бесшовный возврат сетевых адаптеров к исходным настройкам после завершения работы утилиты.
- **Защита от конфликтов:** Превентивная блокировка запуска дублирующих процессов для исключения ошибок маршрутизации.

## ⚙️ Инструкция по эксплуатации

1. **Запуск:** Скачайте и запустите исполняемый файл `WarpBypass.bat`.
2. **Эскалация привилегий:** Утилита автоматически запросит права администратора (UAC), необходимые для модификации сетевых интерфейсов и управления службами.
3. **Инициализация:** В главном меню выберите профиль маршрутизации (рекомендуется профиль `alt12`). При первичном запуске скрипт самостоятельно инсталлирует зависимости.
4. **Статус:** Появление сообщения `Туннель WarpBypass успешно инициализирован` свидетельствует о готовности системы к работе.
5. **Завершение:** Для штатного отключения туннеля и сброса маршрутов закройте окно консоли.

## 🛠 Конфигурация (Клавиша 'S' в главном меню)
Модуль настроек поддерживает следующие параметры:
- **Авто-запуск профиля:** Автоматический старт последней успешной конфигурации без вывода главного меню.
- **Диагностика задержки:** Автоматический опрос заданных хостов после успешной установки соединения.
- **Принудительный сброс DNS:** Очистка системного кэша резолвера (DNS Flush) перед каждым подключением.
- **Авто-обновление:** Фоновая проверка актуальных релизов утилиты и компонентов маскировки в репозитории.

## 🛠 Технологический стек
- **zapret:** Низкоуровневая утилита для DPI-байпасса и модификации сетевого трафика.
- **Cloudflare WARP (CLI):** Консольный клиент для организации защищенного WireGuard-совместимого туннеля.
